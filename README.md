## Web based survey management system

### Installation
  <pre>
  git clone git@github.com:morshedalam/e-survey.git
  bundle install
  bundle exec rake db:setup
  rails s</pre>

### Populate Sample Data
  <pre>
  bundle exec rake db:populate</pre>
